Jika Anda membuka aplikasi ini di perangkat lain, setelan ini tidak terbawa otomatis. Anda TIDAK PERLU membuat Script baru. Cukup tempel (paste) URL yang sudah Anda salin sebelumnya langsung ke kotak di bagian bawah lalu Simpan.

Catatan: Jika ini pertama kalinya Anda menggunakan fitur ini di klinik, ikuti langkah 1 sampai 4 untuk membuat Folder Otomatis di Google Drive Anda.

1. Buat Proyek & Salin Kode

  • Buka web script.google.com (pastikan sudah login akun Google Drive), lalu klik Proyek Baru.
  • Hapus seluruh kode bawaan yang ada di layar putih tersebut.
  • Klik Copy Code di bawah ini, lalu tempel (paste) ke layar tersebut.
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var fileData = data.fileBase64;
    var fileName = data.fileName;
    var mimeType = data.mimeType;
    var folderPath = data.folderPath;

    var decodedFile = Utilities.base64Decode(fileData.split(',')[1]);
    var blob = Utilities.newBlob(decodedFile, mimeType, fileName);

    var rootFolder = DriveApp.getRootFolder();
    var folders = folderPath.split('/');
    var currentFolder = rootFolder;
    
    for (var i = 0; i < folders.length; i++) {
      if(folders[i] === "") continue;
      var folderIterator = currentFolder.getFoldersByName(folders[i]);
      if (folderIterator.hasNext()) {
        currentFolder = folderIterator.next();
      } else {
        currentFolder = currentFolder.createFolder(folders[i]);
      }
    }

    var file = currentFolder.createFile(blob);
    file.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW);

    return ContentService.createTextOutput(JSON.stringify({
      success: true,
      url: file.getUrl(),
      fileName: file.getName()
    })).setMimeType(ContentService.MimeType.JSON);

  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({
      success: false,
      error: error.toString()
    })).setMimeType(ContentService.MimeType.JSON);
  }
}

function doOptions(e) {
  return ContentService.createTextOutput("")
    .setHeaders({
      "Access-Control-Allow-Origin": "*",
      "Access-Control-Allow-Methods": "POST, GET, OPTIONS",
      "Access-Control-Allow-Headers": "Content-Type"
    });
}

2. Terapkan (Deploy) Script

  • Di sudut kanan atas, klik tombol biru Terapkan (Deploy) > pilih Deployment baru.
  • Klik ikon Roda Gigi ⚙️ (Pilih jenis) > pilih Aplikasi Web (Web App).
  • Atur pengaturan: Jalankan sebagai: Saya (Me) | Akses: Siapa saja (Anyone)
  • Klik tombol Terapkan.

3. Otorisasi Izin Akses

  • Klik Otorisasi akses > pilih akun > klik Lanjutan (Advanced) > klik Buka proyek (tidak aman) > Izinkan (Allow).
