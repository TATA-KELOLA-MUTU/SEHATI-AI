Terintegrasi dengan Google Drive Folder Otomatis

Informasi Penggunaan

Pilih elemen penilaian di bawah ini. Jika Anda mengunggah file, sistem akan otomatis memberi prefix (R/D/O) pada nama file dan membuat folder di Google Drive Anda dengan format hierarki:
Bukti_Akreditasi / [Nama Bab] / [Nama Standar] / [Kode EP]

Konfigurasi Google Drive

Buka di Komputer / HP Lain?

Jika Anda membuka aplikasi ini di perangkat lain, setelan ini tidak terbawa otomatis. Anda TIDAK PERLU membuat Script baru. Cukup tempel (paste) URL yang sudah Anda salin sebelumnya langsung ke kotak Hijau di bawah ini lalu Simpan.

Catatan: Jika ini pertama kalinya Anda menggunakan fitur ini di klinik, ikuti langkah 1 sampai 4 untuk membuat Folder Otomatis di Google Drive Anda.

1. Buat Proyek & Salin Kode

  • Buka web script.google.com (pastikan sudah login akun Google Drive klinik Anda), lalu klik Proyek Baru.
  • Hapus seluruh kode bawaan yang ada di layar putih tersebut.
  • Klik Copy Code di bawah ini, lalu tempel (paste) ke layar tersebut.
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var fileData = data.fileBase64;
    var fileName = data.fileName;
    var mimeType = data.mimeType;
    var folderPath = data.folderPath;

    var decodedFile = Utilities.base64Decode(fileData.split(',')[1]);
    var blob = Utilities.newBlob(decodedFile, mimeType, fileName);

    var rootFolder = DriveApp.getRootFolder();
    var folders = folderPath.split('/');
    var currentFolder = rootFolder;
    
    for (var i = 0; i < folders.length; i++) {
      if(folders[i] === "") continue;
      var folderIterator = currentFolder.getFoldersByName(folders[i]);
      if (folderIterator.hasNext()) {
        currentFolder = folderIterator.next();
      } else {
        currentFolder = currentFolder.createFolder(folders[i]);
      }
    }

    var file = currentFolder.createFile(blob);
    file.setSharing(DriveApp.Access.ANYONE_WITH_LINK, DriveApp.Permission.VIEW);

    return ContentService.createTextOutput(JSON.stringify({
      success: true,
      url: file.getUrl(),
      fileName: file.getName()
    })).setMimeType(ContentService.MimeType.JSON);

  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({
      success: false,
      error: error.toString()
    })).setMimeType(ContentService.MimeType.JSON);
  }
}

function doOptions(e) {
  return ContentService.createTextOutput("")
    .setHeaders({
      "Access-Control-Allow-Origin": "*",
      "Access-Control-Allow-Methods": "POST, GET, OPTIONS",
      "Access-Control-Allow-Headers": "Content-Type"
    });
}

2. Terapkan (Deploy) Script

  • Di sudut kanan atas, klik tombol biru Terapkan (Deploy) > pilih Deployment baru.
  • Klik ikon Roda Gigi ⚙️ (Pilih jenis) > pilih Aplikasi Web (Web App).
  • Atur pengaturan berikut:
    • Jalankan sebagai: Saya (Me)
    • Siapa yang memiliki akses: Siapa saja (Anyone)
  • Klik tombol Terapkan.

3. Otorisasi Izin Akses (Sangat Penting!)

  • Google akan meminta izin untuk membuat folder. Klik Otorisasi akses dan pilih akun Anda.
  • Jika muncul layar peringatan "Google belum memverifikasi aplikasi ini", klik tulisan Lanjutan (Advanced) di bagian bawah.
  • Scroll ke bawah dan klik tulisan Buka proyek (tidak aman) / Go to project.
  • Klik tombol Izinkan (Allow).

4. Simpan URL Web App

  • Setelah berhasil, salin tulisan panjang di bawah URL Aplikasi Web.
  • Tempelkan ke dalam kotak hijau di bawah ini dan klik Simpan. Simpan juga URL tersebut di catatan/Grup WA Klinik Anda agar mudah disalin saat buka di komputer lain.
